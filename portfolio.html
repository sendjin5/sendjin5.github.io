Troubleshooting Cases

LMS 인프라 보안 심사 대응


Situation: 공공기관 보안 가이드라인 준수가 필수적인 환경에서 기존 분산된 서버 구조가 보안 심사 기준을 충족하지 못하는 리스크가 존재했습니다.

Action: AWS 환경에서 VPC, Subnet, Security Group 아키텍처를 전면 재설계하고 네트워크 접근 정책을 최소 권한 기준으로 엄격하게 정비하였습니다.

Result: 국가 정보 보안 가이드라인 심사 요건을 완벽히 통과하며, 교육 플랫폼의 운영 안정성 및 보안성을 획기적으로 향상시켰습니다.

공공 데이터 수집 업무 병목 개선


Situation: 산재된 공공 입찰 데이터를 수작업으로 모니터링하며 영업 및 기획팀의 소모적인 반복 업무가 과다하게 발생하고 있었습니다.

Action: 데이터 수집, 가공 및 분석 자동화 파이프라인을 구축하고 이를 시각화하는 전용 관리자 대시보드를 풀스택으로 개발하여 배포했습니다.

Result: 데이터 접근성과 추적성이 비약적으로 향상되었으며, 실무자들의 핵심 데이터 취합 시간을 기존 대비 80% 이상 단축시켰습니다.
